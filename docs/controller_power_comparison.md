# Controller power comparison (ω = 1–4 rad/s)

This file is generated/updated by:

```bash
python3 scripts/controller_power_sweep.py
```

The script runs the full headless controller-period sweep (H = 0.028 m regular waves), writes:
- `output/controller_power_sweep.csv`
- `output/controller_power_sweep.png`
- `docs/controller_power_comparison.md` (this file, with results table + interpretation)

Run the command above in an environment where `build/demo_vgoswec` is available.
